Privacy Policy for Mbot-discord

Last updated: June 22, 2025

  • Data Collected: Server and user IDs for basic operation; command inputs for responding. No messages or personal user data are stored.
  • Use of Data: Only used temporarily to process commands; not shared with third parties.
  • Data Storage: No long-term storage of any user data is performed.
  • Your Rights: If you have questions or concerns, please contact the developer via GitHub Issues or Discord.
